# Getting started

loading from CDN

``` html
<script src="https://cdn.jsdelivr.net/npm/u-spy/dist/index.global.js"></script>
```

## ..and just type on the page

- type `spy` to see spy log dialog.
- type `style` to see ad hoc style editor.
